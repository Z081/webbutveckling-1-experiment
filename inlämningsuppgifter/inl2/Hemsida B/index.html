Vad betyder HTTP och vad är ett komunikationsprotokoll? HTTP är en förkortning av Hypertext Transfer Protocol. Ett kommunikationsprotokoll är en överenskommeslse mellan två eller flera parter om hur man ska kommunicera.
Vad betyder FTP? FTP är en förkortning av File Transfer Protocol och är ett tidigare filöverföringsprotokoll för internet.
Vad kallas HTTPS på fackspråk? HTTPS är en förkortning av Hypertext Transfer Protocol Secure och är ett protokoll för krypterad transport av data.
Vad kallas portal.boden.se på fackspråk? portal.boden.se kallas domän.
Vad kallas "/db/web/external.ns" på fackspråk? "/db/web/external.ns" kallas address
Vad kallas HTML och vad innebär det? HTML är en förkortning av HyperText Markup Language och är ett märkspråk och utgör den grundläggande standarden för World Wide Web(WWW). Webbsidor skrivs i allmänhet som HTML och överförs över Internet med HTTP och HTTPS.
Vad betyder CSS och hur tillämpar man det på hemsidor? CSS är en förkortning av Cascading Style Sheets och är ett språk som beskriver presentationsstilen för ett strukturerat dokument som till exempel typsnitt, textstorlek och färg. CSS tillämpas oftas med hjälp av ett link-element.
